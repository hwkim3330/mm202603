# Collect Next

포상금 확률을 실제로 올리려면 다음만 추가로 모으면 된다.

## 1. 종목 연결 증거
- 종목명 또는 티커가 들어간 채팅 캡처
- "오른다", "정부 프로젝트 수혜", "지금 사라" 같은 문구
- 추천 시각이 보이는 캡처

## 2. 돈의 흐름
- 송금 영수증
- 계좌번호 / 예금주
- 지갑주소 / TXID
- 관리자 승인 메시지

## 3. 운영자 식별
- 텔레그램 ID
- 카카오 오픈채팅 링크
- 전화번호
- 계정명 / 닉네임

## 4. 선행매매 정황
- 추천 전 메시지
- 추천 후 메시지
- 가격 급등 시각
- 운영자 수익 인증/매도 언급

## 5. 제출 시 원칙
- 캡처는 원본 순서 유지
- 파일명에 날짜/시각 넣기
- 타임라인 CSV에 한 줄씩 대응시키기
