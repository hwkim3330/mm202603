# Andyfos Legacy Reference Note

이 문서는 `앤디포스(238090)` 관련 과거 공개 수사·언론보도를 참고자료로만 정리한 부속 메모다.

원칙:

1. 본 부속은 `2026.02.26.~2026.03.09.` 최근 이상거래 신고의 메인 근거가 아니다.
2. 현재 신고의 메인 근거는 한국거래소 공개 경보(`소수계좌 매수관여 과다`, `투자경고 지정예고`)와 최근 공개자료다.
3. 과거 공개 수사·언론보도는 동일 세력 또는 동일 행위가 이번 거래에 직접 연결된다고 단정하지 않는다.
4. 따라서 본 부속은 `재범 위험성 참고` 수준으로만 사용하고, 금감원 신고 본문에는 과도하게 섞지 않는다.

실무 사용 방식:

1. 메인 신고서 본문: 최근 경보와 최근 주가 흐름만 기재
2. 첨부 부속: 본 메모를 추가 첨부할 수 있음
3. 표현: `과거에도 공개 수사 및 언론보도가 있었으므로 반복 위험성 참고 차원에서 별첨함`
4. 금지 표현: `이번에도 같은 세력이다`, `동일범이 확실하다`, `기존 사건의 연장선이다`

추천 부속 문장:

`참고로 본 종목은 과거에도 공개 수사 및 언론보도가 존재하므로 반복 위험성 참고 차원에서 관련 공개자료를 별첨합니다. 다만 본 신고는 최근 한국거래소 공개 경보와 최근 거래 정황을 중심으로 판단해 주시기 바랍니다.`
