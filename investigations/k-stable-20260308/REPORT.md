# k-stable.org 사칭/투자유인 의심 사이트 신고 준비 보고서

작성일: 2026-03-08 (Asia/Seoul)
대상 사이트: https://k-stable.org/
검토 범위: 공개 접근 가능한 페이지와 수동(passive) 기술정보만 수집

## 1. 결론 요약
이 사이트는 공개 페이지 기준으로 다음 정황 때문에 신고 가치가 높다.

1. 금융감독원, 은행연합회, 한국인공지능협회가 직접 설계/감독/실무 참여하는 프로젝트처럼 표시함.
2. 회원가입 과정에서 전화번호, 은행, 계좌번호를 수집함.
3. 서비스 설명에서 예치, 상품 가입, 수익 환원, 투자 현황을 전면에 내세움.
4. 후기 페이지에서 고수익/정기 수익금 수취 문구를 사용함.
5. 약관/개인정보처리방침 링크가 실제 문서가 아니라 `#` 플레이스홀더로 보임.
6. 도메인은 민간 등록 도메인(`k-stable.org`)이며, 공개 검토 시점에 금융감독원 공식 도메인(`fss.or.kr`)과 연결되는 근거를 확인하지 못함.
7. `K-Stable` 명칭은 케이뱅크의 공개 상표 출원 흐름과 혼동될 소지가 있어 민간 금융사 또는 제도권 프로젝트로 오인될 가능성이 있음.

## 3. 사이트 구조 및 의심 포인트
### 3.1 메인 페이지 핵심 문구
홈페이지 원문에서 다음 문구를 확인함.

- `대한민국 정부와 금융감독원이 직접 설계하고 감독하는 은행 제도권 인공지능 참여 프로젝트 K-Stable`
- `프로젝트 기획 및 감독 - 금융감독원`
- `프로젝트 도입 실무 - 은행연합회`
- `프로젝트 개발 실무 - 한국인공지능협회`
- `이재명 정부의 디지털 금융 정책에 발맞춘 프로젝트`
- `금융감독원이 직접 감독하는 공신력 있는 법적 기반 · 국가 신뢰 시스템`

이 문구들은 방문자에게 공공기관 또는 금융규제기관의 공식 참여·감독 사업인 것처럼 인식시킬 소지가 큼.

### 3.2 로그인 / 회원가입 / 투자유인 구조
확인된 경로:
- `/`
- `/service`
- `/reviews`
- `/register`
- `/support`
- `/products`
- `/account`

확인된 기능/경로:
- 로그인 폼: `POST /api/login.php`
- 회원가입 API: `POST /api/register.php`
- 상품가입 API: `POST /api/join_product.php`
- 버튼/메뉴: `투자 상품`, `가입 후기`, `회원가입`, `내 투자 현황`

### 3.3 개인정보/금융정보 수집
`/register` 페이지 자바스크립트 기준 필수 입력 항목:
- `name`
- `phone`
- `username`
- `password`
- `bank`
- `account`

회원가입 성공 메시지:
- `회원가입이 완료되었습니다. 관리자 승인 후 로그인이 가능합니다.`

이는 단순 홍보 페이지가 아니라 계정 생성 및 금융정보 수집 흐름을 갖춘 구조로 보인다.

### 3.4 투자 권유/예치 문구
`/service` 페이지에서 다음 문구를 확인함.
- `K-Stable 예치`
- `안내에 따라 상품에 가입할 금액을 예치합니다.`
- `K-Stable 상품 가입`
- `예치한 금액 내에서 상품 가입을 신청합니다.`

`/reviews` 페이지에서 다음 성격의 문구를 확인함.
- `천만원 투자해서 2달동안 수익금 받았습니다`
- `월초에 수익금이 꼬박꼬박 입금`
- `투자금을 늘릴까 생각중`

이 문구들은 투자권유 또는 유사수신형 유인 정황으로 해석될 수 있다.

### 3.5 기본 신원정보 부재/미흡 정황
캡처한 페이지 기준으로 명시적으로 확인되지 않은 항목:
- 사업자등록번호
- 법인명/대표자
- 사업장 주소
- 정상 동작하는 이용약관 문서
- 정상 동작하는 개인정보처리방침 문서

추가 확인:
- `이용약관`, `개인정보처리방침` 링크는 `#`로 표시되어 실제 문서 연결이 아님.
- `/support`, `/products`, `/account`는 캡처 시점에 사실상 비어 있거나 비정상/미완성 페이지처럼 보였음.

### 3.6 케이뱅크 `K-STABLE` 명칭 혼동 가능성
공개 보도 기준으로 케이뱅크는 `K-STABLE`, `K STABLE`, `KSTABLE` 등 관련 상표를 출원한 바 있다. 따라서 `k-stable.org`가 동일 또는 유사한 명칭을 전면 사용하면, 일반 이용자가 케이뱅크 또는 제도권 금융 프로젝트와 연계된 서비스로 오인할 가능성이 있다.

다만 이 보고서는 공개자료 기준의 혼동 가능성을 지적하는 것이며, 상표권 침해나 케이뱅크 연계 사실을 단정하지는 않는다.
