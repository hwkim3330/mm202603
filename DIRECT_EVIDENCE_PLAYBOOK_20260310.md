# Direct Evidence Playbook 20260310

## 전제

금감원 안내문 기준으로 `주가변동`, `공시자료`, `언론보도` 등 일반에 공개된 자료만으로는 포상금 지급대상이 되기 어렵다.

따라서 다음 단계는 `공개자료형 신고`를 더 늘리는 것이 아니라, 이미 제출했거나 제출 가능한 종목에 대해 `비공개성·직접성 있는 증거`를 붙이는 것이다.

## 지금부터 우선할 증거

### 1. 리딩방 / 오픈채팅 유입 흔적

- 네이버 종토방 이미지에 오픈채팅 링크가 있는지
- 텔레그램 닉네임, 카톡 오픈채팅 코드, QR 이미지
- 동일 링크가 반복 노출되는지

### 2. 반복 홍보 이미지와 OCR

- 같은 문구를 여러 날짜에 반복 사용하는 이미지
- 목표가, 상한가, 세력, 주포, 확정, 보장 수익 문구
- 종목명이 명시된 이미지 우선

### 3. 작성자 반복 패턴

- 동일 닉네임이 같은 종목에서 과장 문구를 반복 게시하는지
- 같은 시간대에 여러 종목에 교차 홍보하는지
- 제목과 본문 논리가 반복되는지

### 4. 게시글과 매매 시점 대응관계

- 게시글 시각 직후 급등/급락이 있는지
- 키움 분봉/틱 데이터와 비교해 시점 표 작성
- 경보 지정 전후로 게시글이 몰리는지

### 5. 삭제 흔적

- 나중에 삭제되거나 비공개 전환된 게시글
- 삭제 전 제목/본문/이미지 확보 여부

## 현재 제출/제출가능 건 중 우선순위

### 1. 인트론바이오(048530)

- 장점:
  - 최근 종토방 과장 문구가 가장 풍부
  - 반복 작성자와 세력/주포/상한가 표현이 확인됨
  - 키움 급등락 데이터도 이미 확보
- 약점:
  - KRX 경보는 계좌집중형보다 투자경고 중심
- 결론:
  - `직접 증거 보강 1순위`

### 2. 퓨쳐메디신(341170)

- 장점:
  - KRX 계좌집중형 경보가 직접적
  - 키움 데이터상 외국인·기관 참여가 거의 없음
- 약점:
  - 최근 종토방 과장 유포는 약함
- 결론:
  - `공식형 신고 1순위`, 직접 증거 보강은 2순위

### 3. 한국피아이엠(448900)

- 장점:
  - KRX 반복 경보가 강함
  - 이미 공개자료형 신고로는 상급 구성 완료
- 약점:
  - 리딩방/이미지형 직접 증거가 약함
- 결론:
  - `제출 완료`, 추가 직접 증거가 있을 때만 재보강

## 하지 말아야 할 것

1. 본인이 쓴 글 또는 유도 글을 증거로 쓰지 말 것
2. 다른 종목을 홍보하는 OCR 본문을 해당 종목 증거로 섞지 말 것
3. 질문형·추정형 글을 허위사실 단정 근거로 쓰지 말 것
4. 공개자료만 더 붙여서 `포상금 보장급`이라고 판단하지 말 것

## 실무 순서

1. 인트론바이오: 반복 작성자/이미지/링크 중심 직접 증거 수집
2. 퓨쳐메디신: 오픈채팅/텔레그램 링크 여부 확인
3. 기존 제출 종목은 추가 직접 증거가 나올 때만 보완 제출
