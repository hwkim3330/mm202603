# Reward Priority Board 20260310

## 기준

- 이미 제출한 종목은 제외
- 금감원 양식 적합성
- KRX 경보 직접성
- 키움 원시데이터 존재 여부
- 온라인 유포 흔적 보강 가능성

## 현재 우선순위

### 1. 한국피아이엠(448900)

- 상태: `제출 권장`
- 이유:
  - `소수계좌 매수관여 과다` 반복
  - `투자경고 지정예고` 반복
  - 키움 원시데이터 보강 완료
  - 기간이 선명함
- 한계:
  - 리딩방/계좌 연결 단서 없음

### 2. 퓨쳐메디신(341170)

- 상태: `제출 가능`
- 이유:
  - `특정계좌(군) 매매관여 과다`
  - `소수지점·계좌`
  - 키움 원시데이터 보강 완료
  - 게시판 본문/OCR 보강 있음
- 한계:
  - KRX 경보 반복성이 한국피아이엠보다 약함

### 3. 인트론바이오(048530)

- 상태: `보류`
- 이유:
  - 파이프라인과 키움 데이터는 있음
  - 다만 현재 드러난 경보가 `투자경고/지정예고` 중심이라 계좌집중형 직접성이 약함

### 4. 흥구석유(024060)

- 상태: `보류`
- 이유:
  - 유가/전쟁 테마로 설명될 여지가 커서 방어 논리가 약함
  - 경보는 있으나 계좌군 직접성은 상대적으로 약함

## 제외 또는 감시 대상

### 케이뱅크(279570)

- 상태: `감시`
- 이유:
  - 보호예수/오버행 루머는 있으나 반복 허위 유포 구조가 선명하지 않음
  - 본인 작성 글 제외 이슈 존재
  - 포상금 메인건으로는 약함

## 실무 순서

1. 한국피아이엠 제출
2. 퓨쳐메디신 제출
3. 인트론바이오·흥구석유는 직접 증거가 더 붙을 때 재검토
