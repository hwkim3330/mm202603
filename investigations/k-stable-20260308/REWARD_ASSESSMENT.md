# Reward Assessment

## Bottom line
Current evidence is strong for impersonation / fraud / unlawful investment inducement reporting.
Current evidence alone is weak for a capital-markets whistleblower reward.

## Why
The Financial Services Commission reward scheme primarily targets capital-markets unfair trading such as:
- market manipulation
- misuse of material nonpublic information
- fraudulent / unfair trading
- market order disruption
- disclosure / reporting violations

A regulator-impersonation website can support a reward-eligible filing only if it is tied to:
- stock-buying inducement
- false rumor dissemination about listed companies
- front-running / leader-room promotion patterns
- coordinated trading / exit evidence

## What would materially improve reward odds
- chat logs connecting `k-stable.org` to a listed stock recommendation
- messages claiming government / FSS backing for a stock or token price rise
- timestamps showing operator holdings before public recommendation
- payment / membership collection for a trading room
- linked Telegram / Kakao / SMS channels promoting securities or token trades

## Current practical route
1. file as impersonation / scam / unlawful investment inducement
2. preserve all evidence
3. separately escalate to FSC / KRX only if stock-linked evidence appears
