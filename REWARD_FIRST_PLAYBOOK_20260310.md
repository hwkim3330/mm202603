# Reward-First Playbook 20260310

## 목적

- 금감원 `증권불공정거래신고` 양식 기준으로 신고 품질을 높이고, 공개자료형 신고를 `포상금 검토선`까지 올리는 작업 기준을 고정한다.
- 핵심은 `많이 신고`가 아니라 `적발 기여도가 높은 신고`를 우선하는 것이다.

## 금감원 양식 기준 핵심 항목

금감원 양식은 아래를 요구한다.

1. 종목
2. 혐의 유형
3. 혐의 기간
4. 불공정거래 정황
5. 위반행위자 또는 관련 계좌·점포 단서
6. 입증자료

따라서 조사도 같은 순서로 해야 한다.

## 포상금 관점 조사 우선순위

### 1. KRX 계좌집중형 경보

가장 우선한다.

- `특정계좌(군) 매매관여 과다`
- `소수계좌 매수관여 과다`
- `단일계좌 거래량 상위`
- `소수지점/계좌`
- `투자경고 지정예고`

위 경보가 반복되면 신고의 뼈대가 된다.

### 2. 키움 원시데이터

공개자료형 신고를 한 단계 올린다.

- 분봉/틱차트
- 호가
- 체결강도
- 투자자/기관/외국인
- 회원사·거래원 흐름

여기서 필요한 건 `수급이 붙었다`가 아니라 `급변`, `반전`, `집중`이다.

### 3. 온라인 유포 흔적

보조자료로 쓴다.

- 네이버 종토방 제목/본문
- 이미지/OCR
- 오픈채팅 링크
- 동일 문구 반복 유포

주의:
- 질문형, 추정형, 의견형 글은 메인 근거로 쓰지 않는다.
- 작성자가 본인인 글은 증거에서 제외한다.

### 4. 공개 기사

기사 자체가 혐의 증거는 아니다.

다만 아래 용도로는 쓴다.

- 기대감 서사와 실질 재료의 괴리
- 확정 수주/실적이 아닌데 급등한 배경
- 시장 해석이 기대 중심인지 여부

## 실제 제출 기준

### 바로 제출 가능한 상태

아래 3개가 있으면 `공개자료형 신고`로 제출 가능하다.

1. KRX 경보 1개 이상
2. 기간이 선명한 본문
3. 첨부 가능한 공개자료 세트

### 포상금 가능성을 올리는 상태

아래까지 붙으면 `기여도`가 올라간다.

1. KRX 경보 반복
2. 키움 원시데이터
3. 온라인 유포 흔적
4. 원문 아카이브와 해시

### 그래도 부족한 것

아래는 공개자료형 신고의 한계다.

1. 리딩방 캡처
2. 오픈채팅 링크
3. 선행매매 정황
4. 특정 계좌·지점과 게시글의 연결
5. 허위 재료와 매매 시점의 직접 대응관계

이 다섯 개가 붙으면 포상금 가능성이 크게 오른다.

## 하지 말아야 할 것

1. 과거 사건을 현재 사건과 증거 없이 억지로 엮지 말 것
2. 종목 여러 개를 한 건에 섞지 말 것
3. 본인이 쓴 글, 유도 글, 의견 글을 메인 증거로 쓰지 말 것
4. 기사만 보고 `조작 확정` 문구를 쓰지 말 것

## 현재 기준 실무 원칙

1. 메인 근거는 `KRX 경보 + 키움 원시데이터`
2. 게시판/기사/OCR은 보조
3. 신고문은 `조사 요청형`으로 작성
4. `포상금 보장`이라는 표현은 내부 판단에도 쓰지 않는다

## 다음 액션 기준

1. 미제출 후보 중 `KRX 반복 경보 + 키움`이 붙은 종목부터 제출
2. 제출 후에는 새 후보를 늘리기보다 `직접 증거`를 확보하는 쪽으로 이동
