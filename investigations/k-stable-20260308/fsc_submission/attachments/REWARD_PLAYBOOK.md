# Reward Playbook

기준일: 2026-03-09

## 1. 목적
이 문서는 `k-stable.org` 건을 단순 사칭/사기 신고에서 끝내지 않고, 가능하면 금융위/KRX의 자본시장 불공정거래 포상금 트랙으로 연결하기 위한 실무 체크리스트다.

## 2. 먼저 선을 긋기
현재 확보된 공개자료만으로는 다음이 강함:
- 기관 사칭 정황
- 투자유인 / 예치 / 수익 홍보 구조
- 개인정보 / 계좌정보 수집 정황

현재 확보된 공개자료만으로는 다음이 약함:
- 상장주 직접 매수 유도 증거
- 허위 재료를 통한 주가 부양 정황
- 운영자 선행매매 / 사후매도 정황
- 불공정거래 유형에 대한 직접 연결

즉, 현재 자료만으로는 `사칭/사기 신고감`은 강하지만, `포상금 확률`은 낮다.

## 3. 포상금 대상 제도와 연결되는 포인트
금융위/KRX 포상금은 보통 다음 유형 적발 기여에서 의미가 커진다.
- 시세조종
- 부정거래
- 미공개정보 이용
- 시장질서 교란행위
- 대량보유/공시 위반 등

따라서 이 사건을 포상금 트랙으로 올리려면, `사이트` 자체보다 다음 연결고리가 필요하다.

## 4. 가장 중요한 추가 증거
### A. 종목 직접 유인
다음 중 하나라도 있으면 급격히 강해진다.
- `케이뱅크 곧 오른다`, `정부 프로젝트 수혜`, `금감원/은행연합회 관련 재료` 같은 메시지
- 특정 상장주 티커/종목명을 명시한 유도 메시지
- 회원가입 또는 예치 후 종목 추천을 제공한다는 문구

### B. 허위 재료 유포
- 정부/금융감독원/은행연합회가 실제 참여하지 않는데 참여하는 것처럼 투자 권유
- `K-STABLE` 명칭을 케이뱅크 공식 프로젝트처럼 표현하며 주식/토큰 투자 유도
- 허위 MOU, 허위 도입, 허위 승인, 허위 라이선스 주장

### C. 선행매매 / 리딩방 구조
- 운영자가 먼저 사고 나중에 회원에게 추천한 정황
- 추천 시각과 주가 급등 시각이 맞아떨어지는 로그
- 추천 후 물량을 넘긴 흔적

### D. 돈의 흐름
- 예치 계좌
- 가상자산 지갑주소
- 거래소 입출금 내역
- 회원비 / 수수료 / 입장료 수취 정황

## 5. 증거 우선순위
### 1순위
- 텔레그램 / 문자 / 카카오톡 캡처
- 음성녹취 / 통화기록
- 종목명 포함 메시지
- 입금 요청 / 수익 보장 문구

### 2순위
- 송금내역 / TXID
- 운영자 프로필 / 닉네임 / 전화번호 / 계정 ID
- 사이트 관리자 승인 메시지
- 계좌 명의자 정보

### 3순위
- 사이트 HTML 원본
- DNS / TLS / RDAP
- 언론기사 및 상표 혼동 자료

## 6. 실제 제출 전략
### 트랙 1: 즉시 신고
- 경찰 ECRM: 사칭 / 사기 / 유사수신 / 투자유인
- KISA: 사칭 / 피싱 / 차단 검토
- 케이뱅크: 브랜드 혼동 및 고객피해 가능성 통보

### 트랙 2: 포상금 준비
다음 중 1개 이상 확보되면 금융위/KRX 제보를 강화한다.
- 종목 직접 추천 메시지
- 허위 재료 유포 메시지
- 리딩방 / 멤버십 / 유료방 구조
- 선행매매 / 사후매도 정황

## 7. 신고서 문장 원칙
써야 하는 표현:
- `~처럼 표시하고 있습니다`
- `~로 오인할 소지가 있습니다`
- `~을 근거로 투자유인이 이루어진 정황이 있습니다`
- `상장주 관련 허위 재료 유포 가능성이 있어 검토 요청드립니다`

피해야 하는 표현:
- `범죄가 확정되었습니다`
- `시세조종이 입증되었습니다`
- `상표권 침해가 확정입니다`

## 8. 현실적 판단
- 현재 단계: `사칭/사기 강함`, `포상금 약함`
- 목표 단계: `상장주 유인 + 허위 재료 + 금전흐름`이 붙으면 `포상금 검토 가치 상승`

## 9. 다음 행동
1. 네가 받은 모든 메시지에서 종목명/티커/수익보장 문구만 따로 추출
2. 계좌번호/지갑주소/송금내역을 사건 타임라인에 붙임
3. 종목 추천 흔적이 확인되면 FSC_TIP_TEMPLATE.txt를 그 내용에 맞게 개정
