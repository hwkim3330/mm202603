# Evidence Matrix

| 항목 | 현재 확보 | 포상금 관점 의미 | 부족한 것 |
|---|---|---|---|
| 기관 사칭 문구 | 있음 | 허위 신뢰 형성 정황 | 이 문구로 종목 유인한 채팅 |
| 예치/상품가입 구조 | 있음 | 투자유인 정황 | 실제 입금 유도 메시지 |
| 계좌정보 수집 | 있음 | 피해 확대 가능성 | 실제 수취 계좌 명의 / 거래내역 |
| 후기의 수익금 홍보 | 있음 | 수익 약속 / 유인 정황 | 종목/토큰 추천 연결 |
| 케이뱅크 K-STABLE 혼동 | 있음 | 브랜드 혼동 / 허위 재료 가능성 | 이를 이용한 매수유도 메시지 |
| 특정 상장주 언급 | 없음 | 포상금 핵심 | 종목명 포함 캡처 |
| 리딩방 / 유료방 구조 | 없음 | 포상금 핵심 | 초대 링크 / 회비 / 가입비 증거 |
| 선행매매 정황 | 없음 | 포상금 핵심 | 시각별 추천 로그 + 가격 흐름 |
| 송금 / 지갑주소 | 개인 증거 필요 | 금전흐름 핵심 | TXID / 입금증 / 예금주 |
| 내부자 / 운영자 신원 | 약함 | 수사 연계 강화 | 전화번호 / 텔레그램 ID / 명의 |
