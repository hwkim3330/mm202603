# InnoRules Reward Gap Note

## 현재 확보한 것
1. KRX 공개 경보: 단일계좌 거래량 상위, 소수지점/계좌, 특정계좌(군) 매매관여 과다
2. 공개 기사: 대표이사 보유주식 증가
3. 네이버 게시판 본문 추출: 주포, 매집, 급등 기대 표현
4. 공개 URL 원문 HTML 보존 및 해시 매니페스트

## 아직 부족한 것
1. 실제 계좌/점포 특정 자료
2. 리딩방·오픈채팅·텔레그램 유인 캡처
3. 추천 직후 급등과 연결되는 비공개성 자료
4. 선행매매 또는 사후매도 정황

## 현실적 평가
1. 현재 패키지는 금감원 양식 적합성이 높다.
2. 조사 필요성도 충분하다.
3. 다만 포상금은 여전히 기여도 문제라서, 공개자료만으로는 유력 단계라고 보기 어렵다.
